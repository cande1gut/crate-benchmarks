==================
CrateDB Benchmarks
==================

A collection of CrateDB_ benchmarks.

Benchmark results are used to compare performance between two different CrateDB
versions.

Current benchmarks:

- The py_ folder contains a set of ``spec`` and ``track`` files for use with
  cr8_.
